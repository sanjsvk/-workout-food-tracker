# Workout Food Tracker

A comprehensive application for tracking workouts and nutrition to help users achieve their fitness goals.

## Features

- Track daily workouts and exercises
- Log meals and monitor nutritional intake
- Set and monitor fitness goals
- View progress analytics and insights

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
